Engine - Block Heater Installation Precautions

Bulletin No: 01-028/09

Last Issued: 07/06/2009

Subject
BLOCK HEATER INSTALLATION CAUTIONS





BULLETIN NOTE

APPLICABLE MODEL(S)/VINS
1999 Miata

DESCRIPTION

When installing a block heater (A), observe the following cautions:





CAUTION
^ Be sure the element (B) is maintained at the 3 O'clock position, as shown, and that it does not rotate during installation. If the block heater rotates while being tightened, damage to the O-ring (C) may occur resulting in a coolant leak. This caution is very important and is not stated in the early version of the installation instructions which were packed with the kit. The instructions in the current production kits have been updated to contain this caution. See illustration above.

^ Be sure to thoroughly lubricate the O-ring (C) only with the grease provided (silicone based) in the kit. Other types of grease will damage the O-ring.
